# Genetics of Brain Size in Drosophila Melanogaster
This repo contains the full project report and working versions of R codes used in GWAS result analysis and curation in addition to codes used in formatting data for usage in [DGRPool](https://dgrpool.epfl.ch/) tool developed by the Laboratory of Systems Biology & Genetics (DePlancke) at EPFL.

## Project Overview
This subproject fits into a larger PhD project of understanding the underlying mechanism and process of learning in Fruit Flies conducted by Samuel Bourgeat [@Sbourgeat](https://github.com/Sbourgeat). The aim of this subproject was to map the genetic componenet of that process through Genome Wide Association and Gene Set Enrichment Analysis using a set of ~200 inbred Fruit Fly Lines.
